1. Overview

This Privacy Policy applies to the AnyDoc's Translator Chrome Extension ("Extension"). It explains how we handle information when you use our Extension. We have designed it to operate with the absolute minimum data access required.

By installing and using the Extension, you agree to the practices described in this policy.

🔍

The Extension accesses only what is strictly necessary to deliver translation:

We do not collect names, emails, IP addresses, browsing history, financial data, or any other personal information.

⚙️

3. How We Use Data

Data accessed by the Extension is used solely for:

  • Translation: Page or selected text is transmitted to your chosen third-party translation API (Google/Microsoft). This happens entirely between your browser and that API.
  • Settings persistence: Your language and UI preferences are saved in browser local storage so your settings are remembered between sessions.

We never use data for advertising, profiling, analytics, or any purpose beyond translation.

🛡️

4. Chrome Extension Permissions

Each permission serves a specific, minimal purpose:

  • activeTab — Access and translate content of the tab you are currently viewing, only when you activate the Extension.
  • storage — Save your preferences locally in Chrome's storage.
  • contextMenus — Add a "Translate Selection" option to the right-click menu.
  • scripting — Inject the translation overlay into the current page.
  • host_permissions (<all_urls>) — Allow translation on any website you choose.

No permission is used for data collection, tracking, or any purpose beyond those listed above.

🌐

5. Third-Party Translation Services

When you request a translation, text is sent to the service you selected. These services have their own privacy policies:

  • Google Translate API — policies.google.com/privacy
  • Microsoft Translator — privacy.microsoft.com/privacystatement

We are not responsible for the privacy practices of these third-party services.

🗑️

6. Data Retention & Deletion

We retain no user data on any server. The only data stored is your preferences in your local browser storage.

To delete all locally stored data:

  • Uninstall the Extension — this clears all local storage, or
  • Use Chrome's built-in extension data management in chrome://settings/siteData.
👶

7. Children's Privacy

Our Extension is not directed at children under the age of 13. We do not knowingly collect any personal information from children. If you believe a child has provided information through our Extension, please contact us immediately.
