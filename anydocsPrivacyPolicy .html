1. Overview

This Privacy Policy applies to the AnyDoc's Translator Chrome extension ("Extension", "we", "us", or "our"). It describes how we handle information when you use our Extension. We take your privacy seriously and have designed our Extension to operate with minimal data access.

By installing and using the Extension, you agree to the practices described in this policy.

🔍

2. What Data the Extension Accesses

The Extension accesses only the data strictly necessary to provide translation functionality:

Data Type Purpose Stored? Shared?
Page text content Sent to translation API to render translated output No Only with chosen translation API (Google/Microsoft)
Selected text Translated on demand when you highlight text No Only with chosen translation API
User preferences Target language, engine choice, UI settings Yes — browser local storage only Never
Active tab URL Required to inject translation script into the current page No Never

We do not collect names, email addresses, IP addresses, browsing history, financial information, health information, or any other personal data.

⚙️

3. How We Use Data

Data accessed by the Extension is used solely for the following purposes:

  • Translation: Page text and selected text are transmitted to a third-party translation API (e.g., Google Translate, Microsoft Translator) chosen by you to produce translated results. This transmission occurs entirely between your browser and the third-party API.
  • Settings persistence: Your preferred language, translation engine, and UI preferences are stored in your browser's local storage so your settings are remembered between sessions.

We do not use data for advertising, profiling, analytics, or any purpose beyond providing translation functionality.

🛡️

4. Chrome Extension Permissions

The Extension requests the following Chrome permissions. Each is used only for the stated purpose:

  • activeTab — Allows the Extension to access and translate the content of the tab you are currently viewing, only when you activate the Extension.
  • storage — Saves your language preferences and settings locally in Chrome's storage so you don't have to reconfigure them each time.
  • contextMenus — Adds a "Translate Selection" option to the right-click context menu for quick translation of highlighted text.
  • scripting — Allows the Extension to inject the translation overlay script into the current page.
  • host_permissions (<all_urls>) — Necessary so the Extension can translate content on any website you choose to translate.

No permission is used for data collection, tracking, or purposes other than those listed above.

🌐

5. Third-Party Translation Services

When you request a translation, the text you wish to translate is sent to the translation service you have selected. These services have their own privacy policies, which we encourage you to review:

  • Google Translate API — https://policies.google.com/privacy
  • Microsoft Translator (Azure Cognitive Services) — https://privacy.microsoft.com/privacystatement

We are not responsible for the privacy practices of these third-party services.

🗑️

6. Data Retention & Deletion

We retain no user data on any server. The only data stored is your preferences in your local browser storage.

You can delete all locally stored data at any time by:

  • Uninstalling the Extension (this clears all local storage for the Extension), or
  • Opening Chrome → Extensions → AnyDoc's Translator → "Clear Data" in the Extension's options page.
👶

7. Children's Privacy

Our Extension is not directed at children under the age of 13. We do not knowingly collect any personal information from children. If you believe a child has provided information through our Extension, please contact us and we will take steps to address it.

Continued use of the Extension after any changes constitutes your acceptance of the updated policy.
