Privacy Policy

AnyDoc's Translator Chrome Extension — your privacy is our priority.

📅 Last updated: April 21, 2025  |   Effective date: April 21, 2025

Overview

AnyDoc's Translator ("the Extension", "we", "our") is a Chrome browser extension that provides translation services for web pages, PDF documents, Word files, images, and YouTube subtitles. This Privacy Policy explains what data the Extension accesses, how it is used, and what is never collected or stored.

By installing and using AnyDoc's Translator, you agree to the terms described in this policy. If you do not agree, please uninstall the Extension.

✅

AnyDoc's Translator does not collect, store, transmit, or share any of the following:

Data Type Collected

Browsing history or visited URLs Never
Keystrokes, clicks, or mouse tracking Never
📄

Website Content Access

The Extension reads the visible text content of the web page the user is actively viewing in order to perform translation. This is the core function of the Extension.

This content is sent directly from your browser to the translation API provider you have selected (Google Translate, Microsoft Azure Translator, or LibreTranslate). The data is processed by that third-party service and the translated result is returned to your browser. The Extension does not store, log, or transmit this content to any server owned by us.

Important: Web page text is sent directly to your chosen translation engine and is governed by that provider's privacy policy. AnyDoc's Translator does not intercept, store, or retain any page content.
💾

Locally Stored Data

The following data is stored locally on your device using chrome.storage.local. It is never transmitted to any external server operated by us.

What is Stored Purpose Location
Target language preference Remember your chosen translation language Device only
Translation engine selection Remember which engine (Google, Microsoft, LibreTranslate) you prefer Device only
API keys (optional) Store your Microsoft Azure or LibreTranslate API key so you do not have to re-enter it Device only
Bilingual and academic mode settings Save display and mode preferences Device only
Translation cache Cache recently translated text to avoid repeated API calls and improve performance Device only
You can clear all locally stored data at any time by going to Chrome Settings, selecting More Tools, then Extensions, then removing the Extension.
🔑

Why We Request Each Permission

Permission Reason
activeTab To read the content of the tab the user is currently viewing when they trigger a translation action
storage To save user preferences, API keys, and translation cache locally on the device
contextMenus To add a right-click option so users can quickly translate selected text on any page
scripting To inject translation logic into pages so visible text can be replaced with translated content
tabs To detect page navigation and identify YouTube pages for subtitle translation
Host permissions (all URLs) Translation is a universal feature and must work on any website the user visits

Third-Party Translation Services

When you use the Extension to translate content, the text is sent to the translation engine you have selected. Each provider has its own privacy policy:

Provider Privacy Policy
Google Translate policies.google.com/privacy
Microsoft Azure Translator privacy.microsoft.com/en-us/privacystatement
LibreTranslate libretranslate.com (open source, self-hostable)

AnyDoc's Translator has no control over how these third-party services process data. We recommend reviewing their privacy policies before use.

🤖

No AI or Machine Learning

AnyDoc's Translator does not use ChatGPT, Gemini, Claude, or any other large language model or AI service. All translations are performed exclusively by the real translation engines listed above. No user data is passed to any AI system.
